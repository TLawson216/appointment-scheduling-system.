# Small Business Appointment Scheduling System
## Author 
Torris Lawson
Cleveland State Student

A web-based application designed to help small businesses manage and organize customer appointments digitally.

## Project Overview

This project is being developed as part of a Software Engineering course.  
The goal is to create a system that allows users to book appointments and view scheduled bookings through a simple web interface.

## Current Features

- Login interface
- Appointment booking page
- Schedule viewing page
- Basic navigation between system pages

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Git & GitHub

## Project Status

This project is currently in the early development stage.  
The current version demonstrates the user interface and application flow.

Future development will include:

- Database integration using SQLite
- Preventing double-booked appointments
- User authentication and session management
- Administrative scheduling tools

A web-based application designed to help small businesses manage and organize customer appointments digitally.

## Installation

1. Clone the repository - https://github.com/TLawson216/appointment-scheduling-system.
2. Navigate into the project directory - cd appointment-scheduling-system
3. Install dependencies - pip install -r requirements.txt
4. Run the application - python app.py
5. Open the application in your browser - After running the application, the terminal will display a message similar to: 'Running on https://###.#.#.#:####' 
Copy and paste the address shown in the terminal into your web browser to open the application.
